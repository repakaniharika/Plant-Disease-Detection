# Plant Leaf Classifier  
**Repakani Harika** — Bio Project 2025  

Live App → https://repakaniharika-plant-leaf-app.streamlit.app  

### What it does  
- Upload any leaf photo  
- Detects **Medicinal Plant** (Tulsi, Neem, Mint, Aloe) or **Common Plant** (Mango, Banana, Grass)  
- Shows **Healthy** or **Diseased**  
- Gives **Uses** and **Cure** (neem oil, sunlight, etc.)  

### My Method  
Uses **SIFT algorithm** + color & texture analysis  
100 % my idea and code
